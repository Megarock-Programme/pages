# Megabots Legal Documents

GitHub Pages untuk Privacy Policy dan Terms of Use.

## Setup GitHub Pages

1. Push folder ini ke repo GitHub (bisa repo terpisah atau branch `gh-pages`)
2. Settings → Pages → Source: Deploy from branch
3. Branch: `main` atau `gh-pages` / folder: `/docs`
4. Save

## URL akan jadi:

```
https://Megarock-Programme.github.io/pages/privacy.html
https://Megarock-Programme.github.io/pages/terms.html
```

## Untuk Notion OAuth Integration

Gunakan URL di atas untuk:
- Privacy Policy URL
- Terms of Use URL
